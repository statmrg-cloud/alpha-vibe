## 1. 프로젝트 소개

Alpha-Vibe는 **AI 기반 투자 분석 + 자동매매 터미널**입니다.

- **Claude AI**가 월가 헤지펀드 전략가 역할로 종목을 분석합니다
- **실시간 주가 데이터**를 Yahoo Finance REST API에서 가져옵니다
- **모의투자** (1억원 가상 자금, KR/US 주식)로 안전하게 연습할 수 있습니다
- 미국 주식 모의투자 시 **실시간 USD→KRW 환율**이 자동 적용됩니다
- **Alpaca Paper Trading**으로 미국 주식 실제 시장 데이터 기반 모의매매가 가능합니다
- **자동매매** 기능으로 AI + 기술적 지표 기반 미국 주식 자동 매수가 가능합니다
- **Alpaca 계좌 패널**을 설정에서 활성화하여 계좌 잔고/포지션을 확인할 수 있습니다

## 2. 시작하기

### 2-1. 필요 환경
- **Node.js** 18 이상 (https://nodejs.org — LTS 버전 권장)
- **npm** (Node.js 설치 시 자동 포함)

### 2-2. 설치

#### Windows 사용자 (초간단)
1. Node.js LTS 설치 (https://nodejs.org)
2. 프로젝트 폴더에서 **`setup.bat`** 더블클릭 → 자동 설치
3. `.env.local` 파일을 메모장으로 열어서 API 키 입력
4. **`start.bat`** 더블클릭 → 서버 시작

#### macOS / Linux 사용자

```bash
# 프로젝트 폴더로 이동
cd alpha-vibe

# 패키지 설치
npm install
```

### 2-3. 환경 변수 설정

프로젝트 루트에 `.env.local` 파일이 있습니다. API 키를 본인 것으로 변경하세요:

```
ANTHROPIC_API_KEY=sk-ant-api03-여기에_본인의_Claude_API_키
ALPACA_API_KEY=여기에_Alpaca_API_Key
ALPACA_API_SECRET=여기에_Alpaca_Secret_Key
ALPACA_BASE_URL=https://data.alpaca.markets
ALPACA_TRADING_URL=https://paper-api.alpaca.markets
```

**API 키 발급처:**
- Claude API: https://console.anthropic.com
- Alpaca Paper Trading: https://app.alpaca.markets (회원가입 후 Paper Trading 키 발급)

### 2-4. 실행

```bash
# 개발 서버 시작
npm run dev
```

브라우저에서 **http://localhost:3000** 을 열면 됩니다.

## 4. 기능별 사용법

### 4-1. AI 투자 분석 (채팅)

#### 사용 방법
1. 하단 입력창에 질문을 입력합니다
2. **Enter** 키 또는 **SEND** 버튼을 클릭합니다
3. AI가 실시간 주가 데이터를 조회한 후 분석 리포트를 제공합니다

#### 질문 예시
| 질문 유형 | 예시 |
|----------|------|
| 종목 분석 | "삼성전자 분석해줘", "AAPL 투자 분석" |
| 종목 비교 | "NVDA vs TSLA 비교해줘" |
| 시장 동향 | "오늘 시장 요약해줘" |
| 투자 전략 | "방어적 포트폴리오 추천해줘" |
| 섹터 분석 | "반도체 섹터 전망은?" |

#### 빠른 질문 버튼
입력창 아래에 있는 버튼을 클릭하면 자동으로 입력됩니다:
- "오늘 시장 요약해줘"
- "삼성전자 분석해줘"
- "AAPL 투자 분석"
- "NVDA vs TSLA 비교"

#### 지원 한국 종목 (한글 입력 가능)
삼성전자, SK하이닉스, 네이버, 카카오, 현대차, LG에너지솔루션, 셀트리온, 기아, 포스코홀딩스, 삼성바이오로직스, 삼성SDI, LG화학, 현대모비스, KB금융, 신한지주, 하나금융지주, 삼성물산, 한국전력, 크래프톤, 미래에셋증권, 한화에어로스페이스, 한화오션, 에코프로비엠, 에코프로, 한미반도체

### 4-2. 모의투자 (Paper Trade) — KR/US 모두 지원

AI 분석 결과에 나타나는 **PAPER TRADE (모의투자)** 섹션의 버튼을 사용합니다.

> **환율 자동 적용:** 미국 주식 거래 시 실시간 USD/KRW 환율이 자동으로 적용됩니다.
> 매수/매도 모달에서 USD 가격과 KRW 환산가가 함께 표시됩니다.
> 포트폴리오에는 원화(KRW) 기준으로 통합 관리됩니다.

#### 매수하기
1. AI에게 종목 분석을 요청합니다
2. 분석 결과 하단의 **[매수]** 버튼을 클릭합니다
3. 매수 모달에서 수량을 입력합니다
   - **MAX** 버튼: 보유 현금으로 최대 수량 자동 입력
   - **10%/25%/50%/100%** 버튼: 최대 수량 대비 비율
4. **[매수 확인]** 버튼을 클릭하면 즉시 체결됩니다

#### 매도하기
1. 보유 종목에 대해 **[매도]** 버튼을 클릭합니다
2. 매도 수량을 입력합니다
3. 예상 손익이 자동 계산되어 표시됩니다
4. **[매도 확인]** 버튼으로 체결합니다

#### 포트폴리오 확인
- 우측 패널의 **PORTFOLIO** 섹션에서 확인
- 총 자산, 수익률, 보유 현금, 보유 종목 수 표시
- **HOLDINGS** 섹션에서 보유 종목 상세 확인 (클릭 시 차트 전환)
- **TRADE HISTORY** 섹션에서 최근 거래 내역 확인

#### 초기화
- PORTFOLIO 카드 우측 상단의 **RESET** 버튼 클릭
- 1억원 초기 자금으로 리셋됩니다

### 4-3. Alpaca 계좌 표시 설정

Alpaca 계좌 패널은 기본적으로 **비활성화**되어 있습니다.

#### 활성화 방법
1. 우측 패널 PORTFOLIO 카드의 **⚙ (설정)** 버튼 클릭
2. SETTINGS 카드에서 **"Alpaca 계좌 표시"** 토글을 ON으로 전환
3. ALPACA ACCOUNT 카드가 나타나고, AI 분석 시 **[실매수] [실매도]** 버튼도 표시됩니다

#### 비활성화 방법
- 같은 설정에서 토글을 OFF로 전환
- 모의투자만 사용할 때는 비활성화 상태로 두면 됩니다

> 설정값은 브라우저 localStorage에 저장되어 페이지를 새로고침해도 유지됩니다.

### 4-4. 실제 매매 (Alpaca Paper Trading)

> Alpaca Paper Trading은 **실제 시장 데이터**를 사용하는 모의매매입니다.
> 실제 돈이 오가지 않지만, 실제 시장가로 체결됩니다.

#### 사용 방법
1. AI 분석 결과의 **REAL TRADE (Alpaca)** 섹션에서 **[실매수]** 또는 **[실매도]** 클릭
2. **Step 1 - 주문 입력:**
   - 주문 유형 선택: 시장가(Market) / 지정가(Limit)
   - 수량 입력 (프리셋: 1/5/10/25/50주)
   - **[주문 확인]** 클릭
3. **Step 2 - 최종 확인:**
   - 주문 내용을 확인합니다
   - **[최종 매수 실행]** 또는 **[최종 매도 실행]** 클릭
4. **Step 3 - 결과:**
   - 주문 ID, 상태가 표시됩니다

> 한국 주식은 Alpaca에서 지원하지 않습니다. 미국 주식만 실매매 가능합니다.

### 4-5. 주가 차트 (프로 차트)

우측 패널 상단의 **PRICE CHART** 에서 확인합니다.

- 보유 종목도 탭에 자동 추가됩니다
- **관심종목 (WATCHLIST)** 클릭 시 해당 종목 차트로 전환

#### 타임프레임
| 버튼 | 설명 |
|------|------|
| **1분** | 1분봉 (당일 데이터) |
| **30분** | 30분봉 (5일 데이터) |
| **60분** | 60분봉 (5일 데이터) |
| **1일** | 5분봉 (당일 데이터) |
| **1주** | 15분봉 (5일 데이터) |
| **1개월** | 일봉 (1개월 데이터) |

| **1년** | 주봉 (1년 데이터) |

#### 줌 & 이동

## 5. 자동매매 설정

우측 패널의 **AUTO TRADE** 섹션에서 제어합니다.

### 5-1. 시작/중지
- **[START]** 버튼 클릭 → 자동매매 시작 (10분 간격)
- **[STOP]** 버튼 클릭 → 자동매매 중지
- 초록색 점 + "ACTIVE" 배지가 보이면 실행 중

### 5-2. 설정 탭

| 항목 | 기본값 | 설명 |
|------|--------|------|
| 건당 최대 금액 | $500 | 자동매수 1건의 최대 금액 (USD) |
| 일일 손실한도 | $1,000 | 이 금액 초과 손실 시 자동 중지 |
| 최소 기술점수 | 30 | 이 점수 이상이어야 AI 분석 진행 |
| 매수 조건 | STRONG BUY만 | BUY 이상으로 변경 가능 |
| 체크 간격 | 10분 | 고정 (변경 불가) |

### 5-3. 관심종목 탭

- 체크박스로 종목별 활성/비활성 토글
- 심볼 + 종목명 입력 후 **[추가]** 버튼으로 새 종목 추가
- **x** 버튼으로 종목 삭제

#### 기본 관심종목
- 루닛 (328130.KS)
- 에이비엘바이오 (298380.KS)
- NVIDIA (NVDA)
- Apple (AAPL)

### 5-4. 로그 탭

자동매매의 모든 활동이 기록됩니다:
- **[CHECK]** — 종목 체크/분석 진행
- **[BUY]** — 자동매수 체결 (초록색)
- **[SKIP]** — 조건 미충족으로 스킵
- **[STOP_LOSS]** — 손실한도 도달 (빨간색)
- **[ERROR]** — 오류 발생 (빨간색)
- **[START/STOP]** — 엔진 시작/중지

### 5-5. 자동매매 동작 흐름

```
10분마다 실행
    │
    ├─ 1. 일일 P&L 확인
    │     └─ 손실한도 초과? → 자동 중지 (STOP_LOSS)
    │
    ├─ 2. 관심종목별 60일 주가 데이터 조회
    │
    ├─ 3. 기술적 지표 분석
    │     ├─ RSI (14)
    │     ├─ SMA (20/50/200)
    │     ├─ MACD
    │     ├─ 볼린저밴드
    │     └─ 거래량 비율
    │     └─ 종합점수 (-100 ~ +100)
    │
    ├─ 4. 점수 ≥ 최소기술점수?
    │     └─ 아니오 → SKIP
    │
    ├─ 5. Claude AI 분석 요청
    │     └─ STRONG_BUY 신호?
    │           └─ 아니오 → SKIP
    │
    ├─ 6. 한국 종목인가?
    │     └─ 예 → SKIP (Alpaca 미지원)
    │
    └─ 7. Alpaca 시장가 매수 실행!
```

> **중요:** 한국 종목(xxx.KS)은 분석만 되고 자동매수는 미국 종목만 가능합니다.

## 6. API 키 관리

### 키 저장 위치
- 프로젝트 루트의 `.env.local` 파일
- 이 파일은 `.gitignore`에 포함되어 GitHub에 업로드되지 않습니다

### 키별 역할

| 키 | 용도 |
|----|------|
| ANTHROPIC_API_KEY | Claude AI 분석 호출 |
| ALPACA_API_KEY | Alpaca 주문/계좌/포지션 조회 |
| ALPACA_API_SECRET | Alpaca 인증용 시크릿 |
| ALPACA_BASE_URL | Alpaca 데이터 API (주가 조회 폴백) |
| ALPACA_TRADING_URL | Alpaca 주문 실행 URL (paper-api) |

### 보안 주의사항
- API 키는 **서버 사이드 (API Route)**에서만 사용됩니다
- 브라우저에는 절대 노출되지 않습니다
- `.env.local`을 Git에 절대 커밋하지 마세요

## 7. 트러블슈팅

### "AI 분석 중 오류가 발생했습니다"
- **원인:** ANTHROPIC_API_KEY가 없거나 잘못됨
- **해결:** `.env.local`에서 키 확인 후 서버 재시작 (`npm run dev`)

### "종목 데이터를 가져올 수 없습니다"
- **원인:** Yahoo Finance 일시 장애 또는 잘못된 종목 심볼
- **해결:** 정확한 종목 코드 사용 (미국: AAPL, 한국: 005930.KS)

### "Alpaca API 키가 설정되지 않았습니다"
- **원인:** ALPACA_API_KEY 또는 ALPACA_API_SECRET 미설정
- **해결:** `.env.local` 확인, Alpaca 대시보드에서 키 재발급

### "Alpaca 주문 실패: forbidden"
- **원인:** Alpaca 계정 미활성화 또는 잘못된 키
- **해결:** https://app.alpaca.markets 에서 Paper Trading 계정 활성화 확인

### 차트가 로딩되지 않음
- **원인:** Yahoo Finance API 속도 저하
- **해결:** 페이지 새로고침 또는 잠시 후 다시 시도

### 포트폴리오가 안 보임
- **원인:** localStorage 초기화
- **해결:** 페이지 새로고침 (데이터는 브라우저에 저장됨)

### 환경 변수 변경 후 반영 안됨
- **해결:** 개발 서버를 반드시 재시작하세요 (`Ctrl+C` 후 `npm run dev`)

## 빠른 테스트 가이드

서버 실행 후 아래 순서로 모든 기능을 테스트하세요:

### Step 1: AI 분석 테스트
1. http://localhost:3000 접속
2. 입력창에 "AAPL 투자 분석" 입력 → Enter
3. AI 분석 리포트가 나오는지 확인
4. 하단에 매수/매도 버튼이 나타나는지 확인

### Step 2: 차트 확인
1. 우측 패널 상단 차트가 로딩되는지 확인
2. AAPL, NVDA, MSFT, TSLA 탭 클릭하여 차트 전환
3. 타임프레임(1분/30분/60분/1일/1주/1개월/3개월/1년/5년) 전환 확인
4. 캔들/라인 차트 전환, 스크롤 줌, 드래그 이동 확인
5. ⚙ 설정 → SMA, CCI, RSI 등 지표 표시 확인

### Step 3: 모의투자 테스트
1. AI 분석 결과의 **[매수]** 버튼 클릭
2. 수량 "10" 입력 → [매수 확인]
3. 우측 PORTFOLIO에 반영되는지 확인
4. HOLDINGS에 종목 추가되는지 확인
5. TRADE HISTORY에 거래 내역 추가되는지 확인

### Step 4: Alpaca 실매매 테스트
1. AI 분석 결과의 **[실매수]** 버튼 클릭
2. 수량 "1" 입력 → [주문 확인]
3. 최종 확인 화면에서 **[최종 매수 실행]** 클릭
4. ORDER SUBMITTED 결과 확인

### Step 5: 자동매매 테스트
1. 우측 패널의 AUTO TRADE 섹션 확인
2. **[START]** 버튼 클릭
3. "로그" 탭으로 전환하여 실시간 로그 확인
4. 종목 분석 진행 로그가 나타나는지 확인
5. 테스트 후 **[STOP]** 버튼으로 중지

### Step 6: 한국 주식 테스트
1. "삼성전자 분석해줘" 입력
2. AI가 삼성전자 시세 데이터를 포함하여 분석하는지 확인
3. 모의투자 매수/매도 가능 확인
